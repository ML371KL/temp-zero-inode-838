# Walk-forward бэктест «Сейсмостанции 21M»

> **Замороженный исследовательский архив.** С 2026-07-19 live-логика зафиксирована как
> `btc-allocation-policy-v1`. Повторный запуск файлов этой директории не меняет live-пороги и не
> является перекалибровкой. Исполняемый контракт и правила выпуска следующей политики описаны в
> [`../POLICY.md`](../POLICY.md).

Инструментарий, на котором проверялась экономическая калибровка панели (аудит 2026-07-14,
переход v2.7.3 → v2.8.0). Реконструирует семейные баллы, блоки, взвешенные шкалы и дерево
гейтов вердикта на каждый день 2018–2026, используя на дату t только данные ≤ t.

## Файлы

| Файл | Назначение |
|---|---|
| `download.mjs` | скачивает полные истории из тех же бесплатных источников, что и панель (FRED CSV, Coinbase-свечи с 2015, Coin Metrics community, The Block ETF, DefiLlama, Blockchain.com, CFTC, Deribit DVOL) в `data/` |
| `engine.mjs` | walk-forward реконструкция всех семейных баллов и режимов → `out/timeline.json` |
| `eval.mjs` | форвардные доходности по режимам, IC семей/блоков, корреляции семей, симуляции стратегий, whipsaw, чувствительность полос/MVRV → `out/report.json` |
| `eval2.mjs` | варианты улучшений: карты позиций, override детекторов, dwell-фильтры, распределение балла, матрица переходов → `out/report2.json` |
| `eval3.mjs` | финальное сравнение v2.7.3 против v2.8.0 на одних данных → `out/final-comparison.json` + `out/chart-data.json` |

## Запуск

```
node backtest/download.mjs   # ~2-3 минуты, сохраняет data/*.json + _manifest.json
node backtest/engine.mjs
node backtest/eval.mjs
node backtest/eval2.mjs
node backtest/eval3.mjs
```

`data/` и `out/` закоммичены по состоянию на 2026-07-14, чтобы результаты аудита были
воспроизводимы без сети; свежий прогон просто перезапишет их.

## Ограничения (важно при чтении цифр)

- Не реконструируемы (нет бесплатной истории): премия США, funding/OI/basis, спред площадок,
  пеги, skew. Эти семьи отсутствуют в бэктесте так же, как отсутствуют в живой панели при
  геоблокировке — покрытие обрабатывается теми же правилами.
- HY OAS (FRED, лицензия ICE) доступен только за ~3 последних года — до 2023-07 семья
  стресса считается из 2 компонентов.
- Форвардные окна перекрываются: «n дней» ≫ эффективного числа независимых наблюдений;
  hit-rate крайних режимов — это единицы эпизодов, а не сотни.
- v2.8.0 калибровалась на той же истории, на которой оценивалась: цифры «после» означают
  «согласуется с данными», а не «гарантировано вне выборки».
